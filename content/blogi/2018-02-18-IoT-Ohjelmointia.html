Virrat Hacklab aloittaa IoT-Ohjelmointikurssin
TIISTAINA 20.2.2018 klo 19:00 - 21:00.
Ilmoittaudu mukaan, enää pari paikkaa jäljellä!

IoT (Internet of Things) -Ohjelmointikurssi on ensisijaisesti lukiolaisilla ja yläasteikäisille tarkoitettu kurssi. Kurssin suorittaminen hyväksiluetaan myös Virtain lukion tietotekniikan kurssiksi TT1.

Kurssin vetää Houston Inc.:ssä työskentelevä Antti Rintala.

Kurssin pääpaino on ohjelmoinnissa, mutta käymme käpi myös muita modernin sovelluskehityksen työkaluja. Jos ohjelmointi kiinnostaa ja varsinkin jos ajattelit tehdä sitä joskus työksesi, tältä kurssilta saat ajantasaiset tiedot siitä, mitä kaikkea täytyy hallita.

Kurssilla rakennetaan saunan lämpömittari tai vaihtoehtoisesti perinteinen sääasema. Valmiin laitteen voi lunastaa itselleen n. 75€ hintaan. Laite kootaan Raspberry PI -tietokoneesta, 7 tuumaisesta kosketusnäytöstä sekä Bluetooth-yhteydellä etäluettavasta RuuviTag:stä.

Käyttöliittymä suunnitellaan ja toteutetaan itse ohjaajien avustuksella ja tutustutaan seuraaviin aiheisiin:

  • Azure
  • Docker
  • JavaScript
  • Python
  • HTML
  • CSS

Tervetuloa mukaan ohjelmoimaan! Ilmoittautumiset Teemu Kontoniemelle teemu.kontoniemi@virtainlukio.fi. Pari paikkaa vielä auki ja jollei lukiosta tai ylästeelta löydy kiinnostuneita, ovat nämä paikat vapaata riistaa!
